
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov

# install sequelize-cli
$ npm install --save sequelize-cli

# Bootstrapping sequelize
$ npx sequelize-cli init

# Running Migrations
$ npx sequelize-cli db:migrate

# Undo Migrations
$ npx sequelize-cli db:migrate:undo

# Running Seeds
$ npx sequelize-cli db:seed:all

# Undoing Seeds sequelize
$ npx sequelize-cli db:seed:undo

```
